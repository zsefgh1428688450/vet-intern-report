## 技术特性

- 纯静态单文件 HTML，零依赖
- 滚动触发的渐入动画（IntersectionObserver）
- 数据计数动画（requestAnimationFrame）
- 固定导航栏 + 阅读进度条 + 返回顶部
- 粒子背景动画
- 响应式设计，移动端适配
- 字体：Noto Serif SC / Noto Sans SC（loli.net CDN 国内优化）
